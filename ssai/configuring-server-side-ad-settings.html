Les configurations publicitaires définissent divers aspects de la lecteur des publicités, notamment les appels publicitaires, le timing, les balises et d'autres options de configuration des publicités. Les paramètres des publicités côté serveur définissent les publicités qui peuvent être demandées pendant les vidéos vidéo à la demande (VOD) et pendant les événements en direct. Les publicités côté serveur tirent parti de la technologie SSAI (Server-Side Ad Insertion) de Brightcove et offrent les principales fonctionnalités suivantes:

  • Étant donné que les publicités sont intégrées au flux vidéo à la demande ou en direct côté serveur, elles ne sont pas affectées par les bloqueurs de publicités
  • Pour les événements en direct, les publicités sont insérées aux points de repère envoyés par votre encodeur ou vous pouvez créer un point de repère instantané à l'aide du module Live Control Room.
  • Vous pouvez configurer des ardoises de remplissage pour remplir tout temps publicitaire non utilisé

Pour en savoir plus sur la publicité côté serveur, consultez Mise en œuvre des publicités côté serveur avec le lecteur Brightcove.

Remarque:
  • Avant de pouvoir utiliser SSAI, votre compte Video Cloud doit être configuré pour Dynamic Delivery et activé pour SSAI. Contactez votre responsable de compte pour plus de détails.
  • VMAP for Brightcove Live SSAI n'est actuellement pas pris en charge.
  • Bien qu'il n'y ait pas de fonctionnalité de prélecteur pour les publicités SSAI, nous déléguons la logique de position de l'annonce au fournisseur d'annonces. La livraison dynamique effectue tous les appels supplémentaires à l'avance. Si nous recevons une annonce du fournisseur de publicité et qu'elle est transcodée dans notre système, nous l'insérons dans le flux.

La page Paramètres des publicités côté serveur est utilisée pour gérer les configurations des annonces en direct et VOD. Notez que pour accéder à la page Paramètres des publicités côté serveur, vous devez être un utilisateur ayant le rôle d'administrateur.

Pour accéder à la page Paramètres des publicités côté serveur:

  1. Connectez-vous à Studio.
  2. Dans la navigation principale, cliquez sur ADMIN, puis sur Paramètres des publicités côté serveur. Une liste de toutes les configurations d'annonces s'affiche.

Pour créer une nouvelle configuration publicitaire, cliquez sur Créer une configuration publicitaire, puis sur le type d'annonce spécifique à créer:

  • Vidéo à la demande (VOD)
  • En direct

Création d'une configuration publicitaire Video On Demand

Remarque: Les configurations publicitaires côté serveur peuvent également être gérées à l'aide de l'API d'insertion de publicités côté serveur. Pour plus d'informations, consultez la rubrique API Ad Config de Video Cloud SSAI.

Pour ajouter une configuration d'annonce vidéo à la demande (VOD), procédez comme suit.

  1. Sur la page Paramètres des publicités côté serveur, cliquez sur Créer une configuration publicitaire > VOD.
  2. Saisissez un nom.
  3. Sélectionnez la réponse du serveur publicitaire:
    • Règles publicitaires DFP
    • VMAP
    • XML intelligent
    • VASTE 3.0
  4. Saisissez l'Ad Tag.
  5. Cochez la case Désactiver les balises côté serveur pour désactiver les balises côté serveur. Lorsque cette option est désactivée, SSAI ne déclenche aucune balise côté serveur et inclut toutes les balises dans la sortie VMAP.
  6. Cliquez sur Enregistrer.

Pour modifier la configuration d'une annonce, cliquez sur le nom de l'annonce, apportez les modifications, puis cliquez sur Enregistrer. Pour supprimer une configuration d'annonce, cochez la case en regard de l'annonce, puis cliquez sur Supprimer. Pour dupliquer une configuration d'annonce, cochez la case en regard de l'annonce, puis cliquez sur Dupliquer.

Une fois les configurations d'annonces VOD créées, ouvrez le module Players et cliquez sur un lecteur pour afficher les propriétés. Lorsque la propriété de publicité côté serveur (SSAI) est activée, une configuration publicitaire peut être sélectionnée. Création d'une configuration de publicité en direct

Remarque: Les configurations d'annonces en direct côté serveur peuvent également être gérées à l'aide de l'API Live. Pour plus d'informations, consultez l'API Live de Brightcove: Insertion de publicités côté serveur (SSAI).

Pour ajouter une configuration de publicité en direct, procédez comme suit.

  1. Sur la page Paramètres des publicités côté serveur, cliquez sur Créer une configuration publicitaire > En direct.
  2. Saisissez un nom.
  3. Sélectionnez la réponse du serveur publicitaire:
    • VASTE
    • DFP (règles publicitaires)
    • XML intelligent
  4. Saisissez l'Ad Tag.
  5. Cliquez sur Enregistrer.

Pour modifier la configuration d'une annonce, cliquez sur le nom de l'annonce, apportez les modifications, puis cliquez sur Enregistrer. Pour supprimer une configuration d'annonce, cochez la case en regard de l'annonce, puis cliquez sur Supprimer. Pour dupliquer une configuration d'annonce, cochez la case en regard de l'annonce, puis cliquez sur Dupliquer.

Une fois les configurations d'annonces en direct créées, lorsqu'un événement est créé avec la propriété Activer l'insertion publicitaire côté serveur (SSAI) activée, une configuration publicitaire peut être sélectionnée dans la Control Room. Pour plus d'informations sur la création d'événements en direct, voir Création et gestion d'événements en direct à l'aide du module Live.

Améliorations du lecteur pour Live SSAI

Les améliorations du lecteur pour Live SSAI de Brightcove vous permettent d'ajouter des composants côté client aux flux en direct, y compris le nombre d'annonces, les comptes à rebours des publicités, les publicités cliquables et les publicités associées. Pour plus de détails, consultez la vue d'ensemble: Document Améliorations du lecteur pour Live SSAI.

Pour utiliser cette fonctionnalité, contactez votre responsable de compte afin d'activer votre compte pour:

  • Diffusions en direct
  • Insertion de publicités côté serveur (SSAI)
  • Améliorations du lecteur pour Live SSAI

Pour ajouter des améliorations au lecteur, procédez comme suit:

  1. Sur la page Paramètres des publicités côté serveur, cliquez sur Créer une configuration publicitaire > En direct.
  2. Saisissez les données comme décrit dans les étapes de configuration de l'ajout d'une annonce en direct ci-dessus.
  3. Sélectionnez Options du client activées pour afficher la liste des options de fonctionnalités.
  4. Sélectionnez les composants côté client que vous souhaitez ajouter à votre flux en direct.
  5. Cliquez sur Enregistrer.
