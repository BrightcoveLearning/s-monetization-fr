Aperçu

Pour créer une configuration de publication, vous devez procéder comme suit:

  • Obtenez l'URL du serveur de balises publicitaires auprès de votre fournisseur de services publicitaires. Il sera utilisé dans Video Cloud Studio pour créer votre configuration publicitaire SSAI.
  • Dans Video Cloud Studio, créez votre configuration publicitaire SSAI. À la fin de la tâche, vous allez copier l'ID du serveur publicitaire pour créer la configuration de votre publicité dans Beacon Classic.
  • Dans Beacon Classic, créez votre configuration publicitaire, que vous pourrez ensuite appliquer à votre contenu vidéo.

Créer un SSAI et un ID de configuration

Suivez ces étapes pour créer une configuration publicitaire SSAI:

  1. Dans {% if site.product_short == Beacon-fr/ ou site.product_short == CorpTV-fr/ %} Studio {% elsif site.product_short == Beacon Classic-fr/ %} Video Cloud Studio {% endif %}, accédez à Admin â†' Paramètres des publicités côté serveur.
  2. Dans le menu déroulant Create Ad Configuration, sélectionnez VOD
  3. Ajoutez un nom pour la configuration de l'annonce.
  4. Sélectionnez la réponse à votre annonce. VMAP ou VAST est recommandé. Si vous utilisez VAST, vous devrez ajouter des points de repère aux vidéos pour spécifier où les coupures de publicité doivent apparaître.
  5. Collez votre balise publicitaire.
  6. Cliquez sur Enregistrer.
  7. Recherchez la nouvelle configuration de votre annonce dans le tableau et copiez l'ID de configuration de votre annonce. Vous aurez besoin de cette valeur pour la configuration de la publicité Beacon.

Création d'une configuration de publication

Pour créer une configuration de publication, procédez comme suit:

  1. Dans Beacon Classic, dans le menu de navigation supérieur, sélectionnez Publicité.
  2. Cliquez sur Ajouter une nouvelle configuration.
  3. Donnez un nom à la configuration publicitaire.
  4. Pour le fournisseur, sélectionnez Brightcove SSAI
  5. Collez l'ID de configuration publicitaire SSAI copié dans les étapes ci-dessus dans l'ID de configuration de la publicité VOD.
  6. Cliquez sur Créer une nouvelle annonce.
