Ce processus de configuration de Live Ad n'est nécessaire que si la chaîne en direct a été activée avec SSAI. Dans le cas contraire, ces étapes ne sont pas obligatoires.

Prérequis

  • Créez une chaîne en direct activée avec SSAI dans Video Cloud.
  • Créez et configurez l'Ad Tag dans Video Cloud.
  • Créez une chaîne dans Brightcove Beacon.
  • Créez la configuration de publication dans Brightcove Beacon.
  • Définissez les droits et la programmation dans la chaîne.

Configuration de la publicité Video Cloud

  1. Dans Video Cloud Studio, sélectionnez le module LIVE.
  2. Créez ou ouvrez le canal SSAI en direct que vous souhaitez utiliser avec Brightcove Beacon.
  3. Dans la Control Room, cliquez sur Publier et intégrer dans le panneau de gauche.
  4. Dans la section Options du lecteur en direct, dans la liste déroulante, sélectionnez la configuration publicitaire que vous souhaitez définir.
  5. Une fois sélectionné, un nouveau code incorporé standard apparaît.
  6. Extrayez l'AdConfigId en copiant une partie de l'URL attribuée à l'AdConfigID, comme indiqué dans l'image.

Création de la publicité dans Brightcove Beacon

  1. Revenez à la plate-forme Brightcove Beacon.
  2. Cliquez sur Publicité.
  3. Créez ou sélectionnez la configuration publicitaire que vous souhaitez définir.
  4. Pour une nouvelle annonce, donnez-lui un nom.
  5. Assurez-vous que le fournisseur est défini sur Brightcove SSAI.
  6. Dans le champ Live Ad Config ID, collez l'URL AdConfigID que vous avez copiée dans la section précédente de ce document.
    Lorsque vous utilisez Beacon Channels, la valeur de l'ID de configuration de la publicité en direct est le jeton de lecteur en direct, au lieu de l'ID de configuration
  7. Cliquez sur Créer une nouvelle annonce.

Définition des droits et planification dans Brightcove Beacon

  1. Dans la barre de navigation supérieure, cliquez sur Chaînes.
  2. Sélectionnez le canal dont vous souhaitez définir la configuration SSAI.
  3. Cliquez sur l'onglet Droits et planification.
  4. Dans Type de droits, sélectionnez Publicité - AVOD.
  5. Dans Configuration de la publication, dans la liste déroulante, sélectionnez la configuration publicitaire que vous venez de créer.
  6. Cliquez sur Créer une nouvelle chaîne pour enregistrer votre travail.
