Aperçu

Un compte Stripe est nécessaire pour autoriser les paiements des clients sur les plateformes suivantes:

  • Web
  • Téléviseur intelligent LG
  • Téléviseur intelligent Samsung
  • Téléviseur intelligent Sony
  • Téléviseur intelligent Toshiba

Cela signifie que Stripe est nécessaire pour l'application Web et toutes les applications Smart TV autres qu'Android TV et Fire TV.

Les étapes suivantes fournissent une vue d'ensemble du processus d'utilisation de Stripe avec Brightcove Beacon:

  1. Créez votre compte en fournissant des informations de base (e-mail, nom, mot de passe).
  2. Activez votre compte en fournissant des informations sur votre entreprise.
  3. Récupérez vos clés API Stripe.
  4. Fournissez au personnel de Brightcove les clés d'API afin qu'elles puissent être ajoutées à Brightcove Beacon Experience.

Création et activation du compte Stripe

  1. Accédez à la page d'accueil de Stripe.
  2. Cliquez sur le bouton COMMENCER MAINTENANT.
  3. Remplissez vos informations de base pour créer le compte.
  4. Vérifiez votre adresse e-mail comme indiqué.
  5. Dans la boîte de dialogue Activer les paiements, cliquez sur Passer pour l'instant.
  6. Sur l'écran d'accueil, sur le côté droit, vous pouvez voir où vous pouvez copier votre clé publiable et votre clé secrète.
  7. Pour obtenir vos clés, vous devrez remplir vos informations personnelles et professionnelles. Pour ce faire, cliquez sur le lien dans le lien de l'infobulle ou cliquez sur le bouton Continuer l'activation en haut à gauche de la page d'accueil.
  8. Une fois que vous avez rempli les informations requises, revenez à la page d'accueil et récupérez vos clés en direct.

Connexion de Stripe à Brightcove Beacon

Une fois que vous aurez vos clés d'API Stripe, vous les transmettrez au personnel de Brightcove qui configure votre Brightcove Beacon Experience. Ils associeront vos clés API Stripe à votre Brightcove Beacon Experience afin que lorsque vous configurez Stripe pour qu'il soit utilisé dans vos applications, il soit associé à votre compte.
