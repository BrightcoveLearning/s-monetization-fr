Étapes pour créer un nouveau coupon

Remarque: Vous pouvez également créer les coupons directement dans Stripe.

Pour créer de nouveaux coupons/bons d'achat dans Brightcove Beacon, procédez comme suit:

  1. Dans Beacon Classic, cliquez sur l'onglet Commerce.
  2. Dans le menu de gauche, sélectionnez Coupons par Stripe.
  3. Cliquez sur le bouton Ajouter un nouveau bon.
  4. Dans le champ Id, tapez le code que les utilisateurs utiliseront.
  5. Pour le champ Nom, utilisez un nom convivial qui peut être affiché sur les factures.
  6. Dans le champ Durée, saisissez la durée du coupon en mois.
  7. Dans le type de réduction, vous avez la possibilité de spécifier la réduction en devise ou en pourcentage. Par exemple, si la réduction est de 3 mois gratuits, définissez la durée sur Répétition avec 3 mois et Type de réduction sur 100 %.
  8. Dans la section Limitations, vous pouvez limiter le nombre d'utilisations ou la date d'utilisation du coupon. Par exemple, supposons que le coupon ne puisse être utilisé que jusqu'à la fin du mois d'août, ou que seules les 100 premières personnes recevront le coupon.
  9. Dans la section Packages/Subscription Associated, spécifiez les abonnements qui peuvent être utilisés avec le coupon. Si vous ne sélectionnez pas d'abonnement, celui-ci s'appliquera à tous les abonnements à des forfaits.
Notez ce qui suit
  • Une fois le coupon/bon créé, tout utilisateur enregistré peut l'appliquer lors de l'achat d'un abonnement.
